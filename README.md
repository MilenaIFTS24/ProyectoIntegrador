# Tío Iroh, Té y Artesanías

## Descripción
**Tío Iroh, Té y Artesanías** es una plataforma web integral diseñada para la gestión operativa y comercial de un emprendimiento dedicado a la venta de tés orgánicos y artesanías únicas. El sistema permite centralizar la administración del inventario, el registro de reservas y la organización de eventos o talleres, ofreciendo una experiencia fluida tanto para el administrador como para el usuario final bajo una estética "Zen".

## Objetivo
El objetivo principal del proyecto es profesionalizar la gestión de **Tío Iroh, Té y Artesanías** mediante la automatización de tareas administrativas, eliminando el desorden en la toma de reservas y brindando una herramienta de control de stock y ventas en tiempo real, garantizando así una operación eficiente y escalable.

## Tecnologías Utilizadas

Este proyecto ha sido desarrollado siguiendo una arquitectura moderna enfocada en el rendimiento y la seguridad:

* **Frontend**: [Angular](https://angular.dev/) - Utilizando componentes independientes (standalone) para una interfaz modular y de alto rendimiento.
* **Backend**: [Node.js](https://nodejs.org/) - Motor robusto para el procesamiento de la lógica de negocio y comunicación con la base de datos.
* **Base de Datos**: [PostgreSQL](https://www.postgresql.org/) - Sistema de gestión de bases de datos relacionales para asegurar la integridad y consistencia de toda la información.
* **Autenticación**: [JWT (JSON Web Tokens)](https://jwt.io/) - Implementación de sesiones seguras y control de acceso basado en roles (Admin/Cliente).

## Características Principales
* **Gestión Administrativa**: Panel de control exclusivo para el administrador con funcionalidades CRUD (Crear, Leer, Actualizar, Borrar) para todas las entidades.
* **Sistema de Reservas**: Calendario automatizado para la gestión de eventos y reservas de productos.
* **Estética Zen**: Diseño visual enfocado en la armonía, la tranquilidad y la experiencia de usuario (UX).
* **Seguridad**: Protección de datos personales mediante políticas de lenguaje claro y autenticación robusta.
